ITW Project – INC 5000 Companies Analysis

This project is based on the Kaggle INC 5000 Companies Dataset, which contains information about the fastest-growing private companies in the United States.
The goal of this ITW project is to clean, analyze, and visualize the dataset to understand industry trends, revenue patterns, growth rates, and geographic distribution.

Dataset

Source: Kaggle – INC 5000 Fastest-Growing Companies
The dataset includes:

Company Name

Industry

Growth Rate (%)

Revenue

Employees

State & City

Rank

Year Founded

Objectives

Clean and preprocess the dataset

Find the average revenue of companies by industry

Identify most and least represented industries

Analyze geographic trends by grouping states into zones

Determine the industries with highest average growth rate

Find top 5 companies with largest increase in staff

Tools & Technologies

Python

Pandas

NumPy

Matplotlib

Seaborn

Google Colab / Jupyter Notebook

Project Workflow

Load and inspect the dataset

Clean missing and inconsistent values

Perform exploratory data analysis (EDA)

Create visualizations

Generate insights and final report

Visualizations

This project includes:

Bar charts

Pie charts

Zone-wise distribution graphs

Growth rate comparison plots

Top 5 employee increase analysis

Folder Structure

/notebooks     → Google Colab or Jupyter notebooks

Conclusion

This project provides meaningful insights into:

Industry performance

Revenue and growth trends

Hiring patterns

Geographic distribution

It demonstrates the use of Python for data analytics, visualization, and business analysis.
